🚀 COMEÇAR AGORA (3 PASSOS):
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  1️⃣  SERVIDOR WEBSOCKET
      $ cd server
      $ npm install
      $ npm start
      
      ✅ Esperado:
      🚀 Servidor WebSocket rodando na porta 8080
         ws://localhost:8080

  2️⃣  APP ANDROID
      $ cd kmp-app
      
      Opção A - Android Studio:
      • File → Open → escolha pasta 'kmp-app'
      • Run → Run 'app'
      
      Opção B - Terminal:
      $ ./gradlew installDevDebug

  3️⃣  TESTAR CONEXÃO
      $ npm install -g wscat
      $ wscat -c ws://localhost:8080
      {"type":"REQUEST_PREFEITURA"}  ← Cole e pressione Enter
      
      ✅ Deve receber JSON com dados da prefeitura

❌ TROUBLESHOOTING RÁPIDO:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  Erro: "Port 8080 already in use"
  → Mude a porta: PORT=8081 npm start

  Erro: "Conexão recusada ws://10.0.2.2:8080"
  → Verifique se servidor está rodando: npm start
  → Se em dispositivo físico, atualize IP em TransparenciaRepository.kt

  Erro: "Gradle sync falha"
  → ./gradlew clean
  → Abra novamente Android Studio

  Erro: "OutOfMemory no Gradle"
  → Aumente heap: org.gradle.jvmargs=-Xmx2048m
